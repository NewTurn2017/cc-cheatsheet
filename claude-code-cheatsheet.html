/ Slash Commands
Session Control
  • /init CLAUDE.md 자동 생성
  • /clear 대화 내용 초기화
  • /compact 대화 압축 (토큰 절약)
  • /compact [키워드] 핵심만 유지하며 압축
  • /resume 중단 세션 재개
  • /rewind 이전 지점으로 되돌림
Monitoring
  • /status 버전 / 모델 / 계정 상태
  • /status line 하단 상태 표시줄 토글
  • /context 토큰 사용 현황 출력
  • /cost 비용 확인
  • /doctor 시스템 상태 진단
Interaction
  • /voice 음성 명령
  • @파일명 특정 파일을 명확히 지칭
Tip: Context Rot 방지 — 장시간 대화 시 /clear 또는 /compact 적극 활용

🔌 MCP & Skills
MCP (Model Context Protocol)
  • API 연동으로 데이터 수집 및 동작 실행
  • 외부 서비스와 Claude를 실시간 연결
  • Claude Code 명령어로 MCP 서버 연결
Webhook
  • 이벤트 기반 자동 실행 트리거
  • 서명 검증으로 보안 강화 필수
Skills (스킬)
  • 반복 작업을 시스템화하는 프롬프트 집합
  • .claude/skills/ 저장 위치
  • skill.md YAML Front Matter 포함
  • Slash /명령어 스킬 호출
  • 자연어 트리거 자동 발동
Sub-agents
  • 스킬 내 하위 에이전트로 작업 분할 위임
